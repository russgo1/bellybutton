Bar Graph:


This graph displays the amount of bacteria found for each of the 10 most populus bacteria in each test subject. OTU ID numbers for each bacterium mark the Y-Axis, and the amount of cultures found mark the X-Axis.





Scrub Gauge:

The gauge above dsiplays the number of times per week each subject's belly button was scrubbed. Washing the belly button may affect the number or type of bacteria found there. The number in the center of the gauge shows the number of scrubs per week. As the number increases, the black bar fills the gauge from left (no scrubs) to right (10 scrubs each week).

Bubble Chart:

This chart displays the relative amounts found of each bacterial culture. Each bubble represents a different culutre. The larger the bubble, the more cultures found. Additionally, more numerous cultures appear higher up the Y-Axis. Each bubble's position along the X-Axis represents the type of bacteria, so that they are spread over the area of the chart. The X-Axis value is the culture's unique ID number.
